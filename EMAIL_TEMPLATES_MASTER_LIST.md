## ğŸ”§ Implementation Guidelines

### Template Creation Checklist
For each new template:

- [ ] Design follows Purra brand guidelines (electric blue gradient, Inter font)
- [ ] Includes UTM tracking parameters
- [ ] Responsive mobile-first design
- [ ] All links include proper campaign name
- [ ] Social media icons (SVG, #0a0f24 color)
- [ ] Correct office address (Raanana, Israel)
- [ ] Unsubscribe link included (where applicable)
- [ ] Contact information up to date
- [ ] Variable placeholders documented
- [ ] Tested across email clients (Gmail, Outlook, Apple Mail)

### Required Variables (All Templates)
```
{{ emailTitle }}       - Email subject
{{ headerTitle }}      - Main header
{{ headerSubtitle }}   - Header subtitle
{{ userName }}         - Recipient name
{{ campaignName }}     - UTM campaign identifier
{{ unsubscribeUrl }}   - Unsubscribe link
```

### Template-Specific Variables
Document additional variables needed for each template in the template file header.
